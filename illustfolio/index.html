Illustfolio

Download Illustfolio View demo

イラストポートフォリオサイトを一瞬で作ろう

Illustfolioは、イラストのポートフォリオサイトに特化した無料Tumblrテーマ（テンプレート）です。
Tumblrに画像をアップロードするだけで簡単にポートフォリオサイトが作れます。
デモを見る

Usage

① テーマをDLして適用する

テーマのHTMLをコピーしてtumblrに適用します。

② Tumblrに画像を投稿

tumblrに「画像」形式で投稿します。

③ 完成

いいかんじになります。

Features

Easy to create

Tumblrに画像を投稿するだけでイラストポートフォリオサイトが作れます。

Costomizable

プロフィールの編集・ソーシャルボタンの有無・固定ページの追加などカスタマイズできます。

Responsive design

ブラウザ幅に合わせて最適化された見た目になります。タブレットやスマートフォンの閲覧にも対応しています。

Retina ready

Retinaディスプレイに最適化されています。アイコンはwebフォントになっており、画面を拡大してもギザギザしません。

Slideshow

新着の投稿画像が自動的にスライドショーに追加されます。タブレットやスマートフォンのスワイプ操作にも対応しています。

Grid design

Twitter Bootstrapのグリッドシステムを使っています。イラストを邪魔しない、シンプルで統一されたデザインです。
